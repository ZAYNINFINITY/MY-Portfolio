# portfolio-web

A clean, modern portfolio website built with React and Tailwind CSS.

## Features

- Hero Section with quick summary
- About Me section
- Projects showcase
- Experience & Education
- Skills & Technologies
- Contact form with EmailJS integration
- Responsive design

## Tech Stack

- React + Vite
- Tailwind CSS
- EmailJS

## Getting Started

```
bash
npm install
npm run dev
```
